# ramdomusers

## Project setup
```
yarn install
```

## Add file .env
visit [RandomAPI](https://randomapi.com/) to read documentation, it is used like this:
```
VUE_APP_API_URL = https://randomuser.me/api/?results=(X)
```
in (X) you can put the number of users that you want to generate.
